# ScopeExit - C++11 scope guard library

# Introduction

## Features
- Easy to use
- Headers only
- Requires C++11 or higher
- No 3rd-party dependencies
- Cross-platform
- Minimal overhead (no std::function or virtual table)
- Clean code
- CMake integration
